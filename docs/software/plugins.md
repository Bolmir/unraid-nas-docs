# Installierte Plugins

> Zuletzt aktualisiert: 2026-04-17

| Plugin | Beschreibung |
|--------|-------------|
| **community.applications** | Unraid App Store (Community Applications) |
| **NerdTools** | Zusätzliche CLI-Tools für Unraid |
| **appdata.backup** | Automatisches Backup des appdata-Shares |
| **user.scripts** | Eigene Shell-Skripte über die Unraid-UI verwalten |
| **rclone** | Cloud-Sync-Tool (rclone) |
| **folder.view2** | Erweiterter Dateimanager in der Unraid-UI |
| **gpustat** | GPU-Monitoring in der Unraid-UI |
| **intel-gpu-top** | Intel GPU Monitoring |
| **prometheus_nvidia_smi_exporter** | Prometheus-Exporter für GPU-Metriken |
| **claude-code** | Claude Code CLI auf Unraid |
